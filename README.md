# siwe-auth-frontned
Frontend in Next.js consuming backend implementing Authentication with SIWE protocol.

## Getting started

### Installation

```bash
$ yarn install
```

### Configure environment

Clone `.env.example` to `.env.local` and configure following environment variables if needed.


### Running the app

```bash
# build
$ yarn build

# start build
$ yarn start
```
